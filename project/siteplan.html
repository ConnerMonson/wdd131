ExoticRides - Website Plan

Site Name

ExoticRides

ExoticRides communicates that this website is about Exotic cars, and its purpose is to show and describe features of the cars.

Site Purpose

The site provides information and images of specific exotic cars, and focuses on making each car look freaking awesome.

Scenarios

What is the fastest super car?

How big is a Lamborghini Aventador's engine?

Color Schema

Background Color: #0B1320

Used for the background everywhere on my site

Accent Color: #E6C068

Used one buttons, hover effects, icons, car highlights, active navigation underline

Secondary Color: #2A1F1A

Used for navigation bar background, cards, section separators, dropdown menus

Text Color: #F3F1EA

Used for all readable Content

Secondary Color: #3A4250

Used on borders, and shadows
